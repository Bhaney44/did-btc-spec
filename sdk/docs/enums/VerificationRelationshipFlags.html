Enumeration VerificationRelationshipFlags

A verification relationship expresses the relationship between the DID subject and a verification method. Each flag corresponds to a bit, and the set of flags form a byte to represent the active verification relationships for a verification method.

See

https://www.w3.org/TR/did-core/#verification-relationships
